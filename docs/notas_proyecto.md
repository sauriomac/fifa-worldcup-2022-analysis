# Notas del Proyecto

## Objetivo
Analizar datos del Mundial FIFA 2022 para generar hallazgos descriptivos y visuales reproducibles.

## Estrategia de trabajo
- Uso de ramas por funcionalidad.
- Integracion via Pull Requests.
- Commits bajo Conventional Commits.

## Pendientes iniciales
- Validar calidad del dataset original.
- Definir diccionario de variables.
- Construir visualizaciones exploratorias clave.
## Estructura del Proyecto
- data/raw: datos originales descargados desde Kaggle.
- data/processed: datos transformados para análisis.
- notebooks: análisis exploratorio, limpieza y análisis final.
- docs: documentación técnica y referencias.
- results: gráficos e informes generados.
## Herramientas Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Git
- GitHub
## Flujo de Trabajo Git
- Crear una rama para cada funcionalidad.
- Realizar cambios locales.
- Utilizar Conventional Commits para registrar cambios.
- Subir cambios mediante Push.
- Crear Pull Request para revisión.
- Integrar cambios aprobados a la rama principal.
